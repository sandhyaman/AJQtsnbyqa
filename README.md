# 手写数字识别系统简介 python1140

## 项目概述

手写数字识别系统是基于Python开发的一套应用，旨在通过机器学习技术实现对用户手写数字的准确识别。系统功能完善，提供了一个用户友好的管理平台。

## 技术栈

- **后端语言**：Python
- **数据库**：MySQL
- **前端技术**：未明确，通常包含HTML、CSS和JavaScript
- **开发工具**：PyCharm
- **Python版本**：3.7

## 功能模块

- 手写字识别
- 手写字管理
- 修改密码
- 个人信息管理
- 用户管理

## 系统角色

系统主要分为以下角色：

- **普通用户**：可进行手写数字的识别和账户信息管理。
- **管理员**：负责用户管理和系统维护。

## 运行环境

- 操作系统：兼容Windows、Linux和macOS。
- Python环境：需配置Python 3.7版本及以上。
- 数据库环境：MySQL服务正常运行。

## 部署步骤

1. 确保Python 3.7及以上版本已安装。
2. 安装依赖库。
3. 配置MySQL数据库。
4. 部署前端文件。
5. 运行后端服务。
